Vue.js
===================
Vue.js
---------------
+ 컴포넌트 기반의 SPA를 구축할 수 있게 해주는 프레임워크

SPA ( Single Page Application )
---------------
+ 단일 페이지 어플리케이션
+ 하나의 페이지 안에서 필요한 영역 부분만 로딩되는 형태
+ 위의 형태덕에 빠른 페이지 변환과 적은 트래픽 양이라는 장점을 가짐.

Vue CLI
---------------
+ 커맨드 라인 인터페이스
+ 터미널을 통해 텍스트 명령어를 통해 실행하는 도구
+ 기본적으로 프로젝트 폴더 구조 등을 자동으로 설정해줌
